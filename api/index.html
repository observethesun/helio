FilesIndex¶

class FilesIndex(*args, **kwargs)[source]¶

Index of files.

Builds a DataFrame containing path to files in columns. By default the DataFrame is indexed by filnames.

kwargsdict of type {name: path}

Path to files that should be indexed and reference name for files collection. Path can contain shell-style wildcards.

Splits index into train and test subsets.

Parameters
